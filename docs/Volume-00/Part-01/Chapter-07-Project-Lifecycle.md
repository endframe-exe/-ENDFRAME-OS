# Chapter 7 — Project Lifecycle

## Purpose
Every project in ENDFRAME OS follows a repeatable lifecycle to improve consistency and knowledge retention.

## Stages
1. Idea Capture
2. Research
3. Planning
4. Execution
5. Testing and Feedback
6. Documentation
7. Iteration
8. Archive

## Deliverables by Stage
| Stage | Primary Deliverable |
|---|---|
| Idea | One-page concept |
| Research | Notes and references |
| Planning | Roadmap and milestones |
| Execution | Working implementation |
| Testing | Issues and validation |
| Documentation | Guides and changelog |
| Iteration | Improvements |
| Archive | Lessons learned |

## Exit Checklist
- Objectives met
- Documentation updated
- Repository organized
- Retrospective completed
- Next actions identified

Following the same lifecycle across projects makes progress measurable and reusable.
