# Chapter 3 — Project Structure

## Repository Layout

```
docs/
  Volume-00/
    Part-01/
templates/
assets/
```

## Purpose

- `docs/` contains handbook content.
- `templates/` stores reusable writing templates.
- `assets/` is reserved for diagrams, illustrations, and media.

This structure is designed to keep ENDFRAME OS organized and scalable as additional volumes and resources are added.
