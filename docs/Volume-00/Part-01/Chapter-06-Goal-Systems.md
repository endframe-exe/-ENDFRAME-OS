# Chapter 6 — Goal Systems

## Introduction
Goals define direction, while systems determine consistent progress. ENDFRAME OS prioritizes building repeatable systems that naturally produce desired outcomes.

## Outcome Goals vs. System Goals
| Outcome Goal | System Goal |
| --- | --- |
| Publish a portfolio | Spend 30 minutes improving it every day |
| Learn a language | Practice vocabulary daily |
| Build a product | Ship one improvement every week |

## Designing Effective Systems
1. Define a clear objective.
2. Break it into measurable habits.
3. Track progress consistently.
4. Review weekly.
5. Refine based on results.

## Weekly Review Questions
- What worked?
- What blocked progress?
- Which habits produced the most value?
- What should change next week?

## Summary
Sustainable systems reduce reliance on motivation and increase long-term consistency.
